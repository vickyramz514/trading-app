# How to Start the Trading System

## 1. Start the app

```bash
pnpm dev
```

## 2. Open in your browser

Use **http://localhost:5173** (not 127.0.0.1)

> If `localhost:5173` fails, try `http://127.0.0.1:5173` — one usually works depending on your system.

## 3. Use an external browser

Open the URL in **Chrome or Safari** (not Cursor's built-in Simple Browser).

---

**URLs:**
- Frontend: http://localhost:5173
- Backend API: http://localhost:3002
